# E-Commerce Data Analysis Project

## Overview
This project analyzes an e-commerce dataset using SQL and Python to extract business insights related to customers, orders, payments, products, and sales trends.

## Technologies Used
- Python
- MySQL
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
The dataset contains information about:
- Customers
- Orders
- Payments
- Products
- Sellers
- Reviews

## Business Questions Solved
- Total revenue generated each year
- Year-over-year sales growth
- Top-selling product categories
- Monthly order trends
- Customer purchase behavior
- Average order value
- Payment method analysis
- Customer retention analysis

## Key Insights
- Revenue increased significantly from 2017 to 2018.
- Credit cards were the most preferred payment method.
- A small number of product categories generated most of the revenue.
- Repeat customers contributed a large portion of sales.
